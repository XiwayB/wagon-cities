# Le Wagon Citites - REACT & REDUX

## Overview
Simple webpage using the following configuration:

React, ReactDOM
Redux
Webpack
HTML
Bootstrap (css only, loaded from a cdn in index.html)


## Features

- Render list of cities
- Click on a city and show the city information
